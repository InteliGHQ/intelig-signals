## Description

Establish GitHub as the trusted source of execution truth for Code Intelligence. This initiative ensures commits and pull requests are reliably ingested and transformed into structured, explainable execution signals. Without this foundation, strategy alignment is speculative and intelligence outputs lack credibility.

## Success Criteria

• Organizations can connect GitHub and select repositories through a simple self-serve setup
• Historical commits and pull requests are fully ingested and verified as indexed
• New commits and PRs are captured automatically via webhooks without duplication
• Git activity is consistently visible across commit listings and Code Intelligence views
• Commits and PRs can be referenced as concrete execution evidence by Strategy and Cognis
